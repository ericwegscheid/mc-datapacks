# How to

* Enable the datapack
`/datapack enable 'file/dev'`

* Run functions from datapack
`/function dev:equip`


## Commands quick refef

player
```
@p = the nearest player to you
@r = a random player
@a = all players in the world
@e = all entities in the world
@s = self
```

clear effects on yourself
```
/effects clear @s
```
